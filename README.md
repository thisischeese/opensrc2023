--------------------------------------------------------------
### 프로젝트 계획 동기
> 요즈음 한국의 개인용 이동장치(PM) 이용자 수가 증가하는 추세이지만 여전히 주행법을 지키지 않는 운전자들은 도로 위에서 쉽게 발견할 수 있다. 

-------------------------------------------------------------
### User Flow
1. 주행 시작과 동시에 실시간 영상 촬영
2. 지면(도로) 객체 탐지 시스템 구조 실행
3. 조건 만족 시 안내문구 출력
4. 운전자 주행 종료
